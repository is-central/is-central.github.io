## 形式

- 3人1チーム
- プレイヤー3: 難易度18
- プレイヤー1と2は、3人の合計難易度が48になるように各自の難易度を選択

## ルール

### 出会い
各チームは以下の秘宝をチームごとに1回までしか選択できません:
- 「憎悪の時代での恨み」
- 「理想の時代への未練」

### ウィシャデル
ウィシャデルは、戦闘後や銀印章による一時的な募集でない限り、選択できません。

### 資金引出制限
- プレイヤー1と2は100源石錐を共有。プレイヤー3は初期50源石錐を引き出せます。
- 制限を超えて引き出した源石錐1つごとに、該当プレイヤーは1源石錐あたり\$500が控除されます。

### 探索準備
- 許可されている3層目のボーナス強化は以下のみ: 電気ケトル、希望+2、計画+2。
- プレイヤーは最初の戦闘をクリアできますが、最初の戦闘でいかなる秘宝も選択してはなりません。
- プレイヤーは2つ目のマスで遭遇を選択しなければなりません。
- 2つ目のマスは「出会い」であり、プレイヤーはこの画面で停止しなければなりません。
- プレイヤーが出会いの秘宝を望まない場合、最初の戦闘後に停止しなければなりません。2つ目のマスの再抽選は許可されます。

### リセット
- 全プレイヤーは利用可能なリセット回数0で開始します。
- エミュレーターのクラッシュ: プレイヤーごとにトーナメント中1回の再開が許可されます（主催者の確認による）。

### その他
- プレイヤーは再抽選後に分隊を変更することが許可されます。
- プレイヤーは探索中、常時自由に通話することが許可されます。
- 秘宝「主なき回想」は探索中に選択することが許可されます。
- プレイヤーは登録に使用したアカウントのみ使用できます。
- すべてのボーナスとペナルティは探索終了時に計算されます。

## トーナメントの仕組み
このトーナメントでは、全プレイヤーが様々な任務を達成してドルを獲得します。プレイヤー1と2はさらに円も獲得します。
プレイヤー3は円ショップで円を使い、探索を有利に進めるためのボーナスを購入できます。
全員の探索終了後、蓄積されたドルはガチャシミュレーターで最終スコアを決定するために使用されます。

### ドル通貨
探索開始時および探索中、ドルは初期分隊とオペレーターをそのコストに応じて購入するために使用されます。
探索中、プレイヤーはさらにドルを獲得し、それをオペレーター購入に使うか、ガチャシミュレーターのために貯めることができます。

- ドルは以下に使用: オペレーター、初期分隊、最終スコア
- ドルは以下を達成することで獲得: バイアスボーナスボーナス、通常ステージ、条件付きステージ、実績、最終所持品数

### 最終スコア計算
- 3人目のプレイヤーの探索終了後、チームは蓄積された全ドルを召喚シミュレーターで使用できます。
- チームの代表者が以下のウェブサイト<a href="https://lungmendragons.com/gachasim" target="_blank" rel="noopener noreferrer">Lungmen Dragons gacha simulator</a>の操作を配信する必要があります。
- チームはシミュレーター内のいずれかのバナーを選択でき、セッション中いつでもバナーを変更できます。
- 「履歴をリセット」がクリックされた場合、その時点から引き続き行われます。リセット前のスコアはVODを確認して決定します。
- 召喚シミュレーターは、ピティなど、ゲーム内の召喚システムと同じルールと確率に従います。
- チームは召喚したオペレーターのレアリティに基づいてスコアを獲得します:
  - 6星オペレーター召喚ごとに = 360 pts
  - 5星オペレーター召喚ごとに = 120 pts
  - 3星および4星オペレーター召喚ごとに = 10pts

## FAQ

**Q**: プレイヤー1と2が同じバイアスボーナスボーナスオペレーターをドラフトした場合、ペナルティドルはどのように計算されますか？\
**A**: バイアスボーナスボーナスオペレーターの基本コストは\$25であり、対応するバイアスボーナスチャレンジを達成した場合にのみ\$0になります。.

ムウィナルがバイアスボーナスボーナスオペレーターである場合のシナリオを考えます:
1. プレイヤー1がムウィナルボーナスチャレンジを達成し、プレイヤー2がムウィナルをドラフトするがチャレンジを達成しない場合、プレイヤー2に発生するコストは\$25 x 3 = \$75
2. プレイヤー1がムウィナルボーナスチャレンジを達成せず、プレイヤー2がチャレンジを達成する場合、プレイヤー1に発生するコストは\$25
3. プレイヤー1と2の両方がチャレンジに失敗した場合、発生するコストは\$25 + \$75 = \$100
4. プレイヤー1と2の両方がチャレンジを達成した場合、両者のコストは\$0。ただし、ボーナスは1回のみ請求可能 (\$150 x 1)

**Q**: 円ショップのオペレーター制限ペナルティ解除はどのように機能しますか？\
**A**: オペレーター制限ペナルティ解除報酬はプレイヤー3にのみ適用され、実質的にオペレーターのコストを基本コストに戻します。プレイヤー1と2はこの効果の影響を受けません。

**Q**: トリプルエンディングとは何ですか？\
**A**: トリプルエンディングとは、3体のエンディングボスが撃破された集成戦略の探索です。エンディングボスには5層、6層、7層の悪路凶敵ステージが含まれます。「勝利の反響」のみのクリアはエンディングボスの撃破としてカウントされません。

**Q**: 「3人のプレイヤー間で、3つの異なる悪路凶敵ステージにおいて、同じ3体のオペレーターをボートに乗せ...」というチーム実績について、何がカウントされる/されないかの例はありますか？\
**A**: 以下のシナリオを考えます:
1. プレイヤー1が「謁見」でボートにディープカラー、メイ、ビーンストークを乗せる。プレイヤー2と3もそれぞれ「説法」と「運命の終幕」で同じオペレーターをボートに乗せる。実績は達成されたとみなされる。
2. プレイヤー1が「謁見」と「説法」でボートにディープカラー、メイ、ビーンストークを乗せる。プレイヤー3も「運命の終幕」で同じオペレーターをボートに乗せる。実績は達成されたとみなされる。
3. プレイヤー1が「謁見」と「説法」でボートにディープカラー、メイ、ビーンストークを乗せる。プレイヤー2と3も「説法」で同じオペレーターをボートに乗せるが、「運命の終幕」では乗せない。実績は未達成とみなされる。
4. プレイヤー1が「謁見」でボートにディープカラー、メイ、ビーンストークを乗せる。プレイヤー2と3もそれぞれ「説法」と「運命の終幕」で同じオペレーターをボートに乗せる。しかし、「運命の終幕」でボートが青い枠に到達しない。実績は未達成とみなされる。

**Q**: 初期招集時の初期6★オペレータードラフトは、初期資金のドルを消費しますか？\
**A**: いいえ、初期オペレータードラフトの選択は、初期分隊を選択する能力にのみ影響し、初期資金には影響しません。残りの初期資金はドルに変換されます。

**Q**: ドルのボーナス/ペナルティはいつ計算されますか？\
**A**: ドルのボーナス/ペナルティは、プレイヤーの個々の探索終了時に計算されます。

**Q:** アミィリストのオペレーターは、他の6★オペレーターと同じように価格が上がりますか？\
**A:** いいえ、5★のアミィリストオペレーターは、チーム内で何人のプレイヤーが選択しても$35のままです。
